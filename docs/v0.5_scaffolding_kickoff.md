# v0.5 Scaffolding Kickoff

v0.5 will focus on chromosome-scale scaffolding with clear evidence and conservative review.

## Goals

- Add YaHS Hi-C scaffolding workflow.
- Add 3D-DNA/Juicebox/JBAT visual curation workflow.
- Add reference-guided scaffolding notes with reference-bias warnings.
- Add AGP generation and validation guidance for scaffolded assemblies.
- Add Hi-C contact map QC checklist.
- Add post-scaffolding release validation expectations.

## Initial Decision Points

Before scaffolding, decide:

- Is Hi-C available and high quality?
- Is a close reference available?
- Is reference-guided scaffolding acceptable for the project goal?
- Are contigs already screened for contamination and organelles?
- Are candidate misassemblies reviewed before scaffolding?
- Will scaffolded chromosomes be submitted with AGP?

## Rule of Thumb

Scaffolding should improve chromosome-scale organization without hiding unresolved sequence, contamination, haplotig duplication, or real structural variation.
