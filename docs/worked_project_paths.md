# Worked Project Paths

This page collects concrete beginner-friendly routes through the protocol for common crop assembly situations.

Use these paths when the full docs tree feels too open-ended and you want a more opinionated starting sequence.

## Available Worked Paths

- [Diploid inbred HiFi-only path](case_paths/diploid_inbred_hifi_only.md)
- [Heterozygous HiFi plus Hi-C path](case_paths/heterozygous_hifi_hic.md)

## How To Use These Paths

Each worked path is meant to answer:

- what to do first
- what to postpone
- what outputs to keep
- what evidence should trigger caution
- which later sections matter most

They do not replace the rest of the protocol. They help you enter it with a realistic first-pass strategy.
