# QC

This section is for deciding whether the assembly is merely finished or actually believable.

## Read This Section First If

- you need to interpret contiguity, completeness, and k-mer evidence together
- you want to decide whether a suspicious contig should be reviewed, not just deleted
- you are checking contamination, organelles, or unexpected sequence classes
- you are preparing figures or summary tables for internal review, a manuscript, or submission

## Best Starting Pages

- [Assembly metrics and interpretation](assembly_metrics.md)
- [Dotplots for assembly review](dotplots.md)
- [Contamination review](contamination.md)
- [Contamination decision workflow](../contamination_workflow.md)
- [QC figures](../qc_figures.md)

## Practical Outcome

After this section, you should know:

- whether the assembly is structurally plausible
- whether contamination or organellar sequence needs action
- which contigs or scaffolds deserve manual curation
- which plots and tables should follow the project into review and release

## Read Next

If structural problems remain, continue to [Curation](../curation/index.md). If QC is clean and you are moving toward chromosome scale, continue to [Scaffolding and Finishing](../scaffolding/index.md).
