# Research Notes

These pages publish working research notes used to ground protocol decisions in the
HiFi genome assembly documentation.

They are not polished step-by-step protocol pages. Use them as transparent source
material for why the main workflow defaults to particular methods, where alternate
methods fit, and which claims need stronger evidence.

## Published Notes

- [HiFi genome assembly methods deep research pass](2026-04-27_hifi_methods_deep_research.md)
- [Recent crop genome publication methods pass](2026-04-27_recent_crop_publication_methods_pass.md)

## How To Use These Notes

- Use the crop-publication pass to check claims about what recent high-profile
  crop genome papers are actually choosing.
- Use the deep methods pass to find supporting protocols, tool docs, publications,
  community discussions, missing methods, and follow-up work.
- Treat both notes as dated snapshots. Tool versions, vendor recommendations,
  database policies, and community best practices should still be checked before a
  new release or manuscript methods section is finalized.
